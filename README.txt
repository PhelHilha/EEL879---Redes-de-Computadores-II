2. DESCRIÇÃO DO PROJETO
-----------------------

Este projeto consiste em um website informativo e multi-página desenvolvido como parte da avaliação da disciplina de Redes de Computadores II. O site apresenta um estudo detalhado sobre a tecnologia LoRaWAN, abordando desde seus conceitos fundamentais e arquitetura até análises técnicas de desempenho, segurança, casos de uso práticos e uma seção interativa de questões.

O objetivo é apresentar o conteúdo de forma clara, organizada e visualmente agradável, utilizando uma estrutura de navegação intuitiva que separa o trabalho em seções temáticas.


3. ESTRUTURA DE ARQUIVOS
------------------------

O projeto está organizado nos seguintes arquivos, que devem estar na mesma pasta:

   /
   |-- style.css       (Arquivo de estilo principal para todas as páginas)
   |
   |-- index.html      (Página inicial / Home)
   |-- introducao.html (Página sobre a Introdução ao LoRaWAN)
   |-- camada-fisica.html (Página sobre a Camada Física / Tecnologia LoRa)
   |-- protocolo.html  (Página sobre o Protocolo LoRaWAN)
   |-- desempenho.html (Página sobre Desempenho e Escalabilidade)
   |-- seguranca.html  (Página sobre Segurança no LoRaWAN)
   |-- casos-de-uso.html (Página sobre Aplicações e Casos de Uso)
   |-- questoes.html   (Página interativa de Perguntas e Respostas)
   |-- bibliografia.html (Página com as Referências Bibliográficas)


4. COMO EXECUTAR O PROJETO
--------------------------

Para visualizar o site:

   1. Certifique-se de que todos os arquivos listados acima estão na mesma pasta.
   2. Abra o arquivo `index.html` em qualquer navegador de internet moderno (Google Chrome, Mozilla Firefox, Microsoft Edge, etc.).
   3. A navegação entre as diferentes seções do trabalho é feita através do menu no topo de cada página.

   Observação: É recomendável ter uma conexão com a internet para que a fonte "Poppins" (do Google Fonts) seja carregada corretamente, garantindo a melhor experiência visual.


5. TECNOLOGIAS UTILIZADAS
-------------------------

   - HTML5: Para a estruturação do conteúdo de todas as páginas.
   - CSS3: Para a estilização visual, layout e responsividade.
   - JavaScript: Para a funcionalidade interativa da seção "Questões".
